## Qu'est-ce que c'est ?

La Bible Melvynx est une compilation exhaustive de TOUT ce que
Melvynx (developpeur francais, createur de Umail.io, SaveIt.Now,
WhisperTurbo) a enseigne sur Claude Code a travers ses videos YouTube,
sa formation premium, et son repo GitHub AIBlueprint.

Ce document sert de **reference absolue** : avant toute question sur
Claude Code, on consulte la Bible. Si la reponse y est, on l'applique
directement.

## Installation

### Methode 1 : Comme reference dans Claude Code
```bash
# Copier la Bible dans la config Claude Code
cp BIBLE_MELVYNX_V5.md ~/.claude/bible/BIBLE_MELVYNX_COMPLETE.md

# Creer une regle de reference
cat > ~/.claude/rules/30-bible-melvynx-reference.md << 'EOF'
# Bible Melvynx — Source de verite
Fichier : ~/.claude/bible/BIBLE_MELVYNX_COMPLETE.md
Consulter AVANT toute question sur Claude Code.
EOF
```

### Methode 2 : Comme CLAUDE.md projet
```bash
# Dans un projet specifique
cp BIBLE_MELVYNX_V5.md ./CLAUDE.md
```

### Methode 3 : Lecture seule
Ouvrir `BIBLE_MELVYNX_V5.md` et chercher par chapitre.
