# Bible Melvynx — Deploiement Rapide

## Netlify Drop (demos, prototypes)
1. Demander a Claude : "prepare un dossier pour deployer sur Netlify Drop"
2. Claude genere un dossier `dist/`
3. Aller sur `app.netlify.com/drop`
4. Drag & drop le dossier `dist/`
5. URL publique instantanee

## Vercel (production)
- Pour les projets Next.js et React
- Deploiement automatique depuis GitHub
- Stack Melvynx : Vite + React + ShadCN + Tailwind (ou Next.js)
